## INTEGRATION STEPS

    - Clone the project:

        git clone git@github.com:kaizen-squad/upply-backend.git


    - Install dependencies:

        composer install

        npm install


    - Create the environment variables file:

        cp .env.example .env


    - Generate the application key:

        php artisan key:generate

## DOCKER USAGE

To run the application using Docker, use the following command:

    docker-compose up -d --build

> **Note:** When switching back to a local environment (`php artisan serve`), ensure your `.env` file uses `DB_HOST=127.0.0.1` and clear any cached absolute paths generated by Docker:
> 
>     sudo rm -f storage/logs/laravel.log bootstrap/cache/*.php
>     php artisan optimize:clear
